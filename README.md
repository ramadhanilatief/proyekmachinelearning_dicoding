# Machine Learning Project - Dicoding Indonesia

## Insight
In this project, I obtained several important insights from both the clustering and classification processes:
1. The K-Means clustering model successfully grouped customers into 2 distinct clusters, indicating that there are clear differences in customer transaction behavior and financial profiles.
2. The clustering model achieved a Silhouette Score of 0.568, which indicates a reasonably good separation between clusters and suggests that the generated clusters are meaningful and interpretable.
3. The clustering analysis revealed two main customer segments:
  - Cluster 0: Customers with moderate transaction amounts, relatively higher account balances, and professional working backgrounds.
  - Cluster 1: Customers with lower account balances and different transaction behavior patterns, representing a distinct customer group.
4. The classification models (Decision Tree and Random Forest) achieved 100% accuracy, precision, recall, and F1-score, demonstrating that the cluster labels generated by K-Means can be predicted very effectively using the available customer attributes.
5. Hyperparameter tuning on the Decision Tree model did not significantly improve performance because the model had already achieved perfect classification results before tuning.
6. Customer attributes such as transaction amount, account balance, customer age, and transaction duration play an important role in distinguishing customer segments and contribute significantly to the clustering and classification processes.

## Conclusion
The combination of clustering and classification techniques successfully identified meaningful customer segments and produced highly accurate predictive models. These results can support customer segmentation strategies, personalized marketing campaigns, and more targeted financial service offerings.
